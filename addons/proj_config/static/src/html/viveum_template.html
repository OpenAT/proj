Qualitätskriterium

Bereits seit 2002 wird Pro Juventute durchgehend das Österreichische Spendengütesiegel verliehen. Das Österreichische Spendengütesiegel ist ein Qualitätsmerkmal der Non-Profit-Organisationen.
